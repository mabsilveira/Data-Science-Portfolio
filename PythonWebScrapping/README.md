# Newspaper Web Scraper

A simple project to pull opinion articles off a newspaper website

Using the requests-html module to access the newspaper website  and retrieve all recent opinion articles from a select list of authors

The script then compiles the articles in a pdf file and sends it via email

### Contents
* __newspaper-webscraper.py__: Simple web scraper using Python's requests-html
* __requirements.txt__: Requirements file for virtual environment
